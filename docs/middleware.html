Middleware

Express-style middleware for annotify. Runs on every request, with chain termination via next(). Distinct from useInterceptor, which runs once at build() time.

Quick start

import { AppBuilder, staticFiles } from 'annotify';

const app = new AppBuilder();

// Global middleware (every request)
app.use((req, res, next) => {
  console.log(req.method, req.url);
  next();
});

// Path-prefixed middleware (only fires for matching paths;
// req.url is rewritten to drop the prefix).
app.use('/static', staticFiles('./public'));

Per-route middleware with @Use

Annotate individual methods with @Use(...middlewares). The chain runs after route match, before resolveArgs. 

import { Controller, GetMapping, Use } from 'annotify';

const requireAuth: MiddlewareFn = (req, res, next) => {
  if (!req.headers['x-token']) {
    res.writeHead(401); res.end(); return;
  }
  next();
};

@Controller('/api')
export class ApiController {
  @GetMapping('/public')
  public() { return { ok: true }; }

  @GetMapping('/private')
  @Use(requireAuth)
  private() { return { secret: 42 }; }

  // Multiple middlewares stack top-to-bottom.
  @Use(requireAuth, rateLimit(60))
  @PostMapping('/submit')
  submit() { return { ok: true }; }
}

Ordering

The pipeline runs in this order for every request:

  1. Pre-routing middleware — the global app.use(...) chain. Path-prefixed entries are filtered to those whose prefix matches the request path. req.url is rewritten so downstream code sees the mount-relative path.
  2. Route match — the trie lookup against @GetMapping / @PostMapping / etc.
  3. Per-route middleware — the @Use(...) chain on the matched method (plus class-level middleware from @UseClass(...)).
  4. Handler invocation — resolveArgs then the bound handler.
  5. Post-routing fallthrough — if no annotation route matched, the global middleware chain runs once more. If nothing writes a response, the framework sends 404 Not Found.

Built-in middleware

import {
  json,          // JSON body parser (alternative to framework's built-in)
  corsMw,        // app-wide CORS headers
  requestLogger, // stdout access logger
  staticFiles,   // serve a directory under a mount prefix
} from 'annotify';

app.use(corsMw({ origins: ['https://my.app'], credentials: true }));
app.use(json({ limit: '2mb' }));
app.use(requestLogger());
app.use('/static', staticFiles('./public', { index: 'index.html' }));

Static files

import { staticFiles } from 'annotify';

// Mount at root, serve from ./public
app.use(staticFiles('./public'));

// Mount under /static
app.use('/static', staticFiles('./public'));

// With options
app.use('/static', staticFiles('./public', {
  index: 'index.html',         // default
  cacheControl: 'public, max-age=3600',
}));

Static middleware honors Range requests, emits ETag + Last-Modified + If-None-Match for cheap 304s, and rejects path-traversal attempts (..).

Response helpers

import { html, redirect } from 'annotify';

@GetMapping('/page')
page() { return html('<h1>plain</h1>'); }

@GetMapping('/go')
go() { return redirect('/page'); }

Settings

app.set(key, value) and app.get(key) form a simple settings bag. Currently used for template configuration: 

app.set('views', './views');         // template directory
app.set('view engine', 'html');     // extension + engine name
